# Quantix Faucet

## Project Info

**URL**: [Quantix Faucet](https://faucet.quantix.network)

## Getting Started

This project is built with Vite, React, TypeScript, shadcn-ui, and Tailwind CSS.

### Prerequisites

- Node.js & npm installed

### Installation

```sh
# Clone the repository
git clone <YOUR_GIT_URL>

# Navigate to the project directory
cd quantix-stake-faucet

# Install dependencies
npm install
```

### Development

Start the development server:

```sh
npm run dev
```

### Build

Build for production:

```sh
npm run build
```

## Technologies

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
